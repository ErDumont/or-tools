RoutingModel::ResourceGroup

Detailed Description

A ResourceGroup defines a set of available Resources with attributes on one or multiple dimensions.

For every ResourceGroup in the model, each (used) vehicle in the solution must be assigned to exactly 1 resource, and each resource can in turn be assigned to at most 1 vehicle. This vehicle-to-resource assignment will apply the corresponding Attributes to the dimensions affected by the resource group. NOTE: As of 2021/07, each ResourceGroup can only affect a single RoutingDimension at a time, i.e. all Resources in a group must apply attributes to the same single dimension.
